Support Information

VelocityOne FS2024 Configurator is a local browser-based tool for configuring the Turtle Beach VelocityOne Flight yoke and quadrant for Microsoft Flight Simulator 2024 on PC.

Delivery

The product is sold through Gumroad checkout and delivered as a digital ZIP download. The package includes the local browser app, runtime files, assets, README, and license. No account system or manual setup service is required.

Requirements

  • Turtle Beach VelocityOne Flight yoke and quadrant.
  • Microsoft Flight Simulator 2024 for PC.
  • A modern desktop browser.

Tested Environment

  • Windows 11.
  • macOS 26 for the configurator UI and profile export.
  • Microsoft Flight Simulator 2024 for PC, Sim Update 4 / 1.6.32.0.

Basic Setup

  1. Keep index.html, assets/, and vendor/ in the same folder.
  2. Double-click index.html.
  3. The configurator opens locally in your browser.

Importing Profiles

Before importing any exported XML files, back up your existing Microsoft Flight Simulator 2024 controller profiles. After exporting from the configurator, copy the generated XML files to your simulator controller profile folder and load them from the simulator's Controls settings.
