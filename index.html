VelocityOne FS2024 Configurator

A visual browser-based configurator for the Turtle Beach VelocityOne Flight yoke and quadrant. Build bindings, save profiles, export XML files, and generate a PDF reference sheet without hand-editing controller profiles.

What It Does

Want to see the workflow first? Watch the demo video on YouTube.

Visual Binding Workflow

Click directly on the yoke or quadrant diagram instead of guessing where a control lives.

Profiles and Presets

Create presets, save your current setup as a JSON profile, and load it again later.

FS2024 XML Export

Export the four XML files Flight Simulator needs for yoke/quadrant general and airplane controls.

Conflict Awareness

Surface duplicate action assignments before export so problems are easier to catch.

Focused Views

Filter the setup between General bindings, Airplane bindings, or both.

PDF Reference Sheet

Generate a printable reference sheet for quick lookup outside the configurator.
